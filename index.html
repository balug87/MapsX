    About MapsX

    MapsX applies fully stylized “filters” — color palette, typography and point-of-interest icons — to real map data, and exports a selected area as a print-ready PNG.

    It uses OpenStreetMap vector tiles served by OpenFreeMap (no API key required), rendered with MapLibre GL JS. Search is powered by Nominatim. Fonts are rendered into map glyphs locally in your browser, so any bundled font can appear on the map itself.

    If you print or publish an export, keep the OpenStreetMap attribution visible.
